# Tricle

> Three elements. One cycle.

가위바위보에서 착안한 카드 전략 게임. 불/물/풀 3원소 상성 + 포커 베팅 구조.

🔗 **[Play Now](https://inoocap-ux.github.io/tricle)**

---

## 핵심 메카닉

- **비대칭 3세트** (불3물1풀1 / 불1물3풀1 / 불1물1풀3) — 상대 구성 비공개
- **라운드마다 세트 교체 가능** — 블러핑 요소
- **순차 베팅** → 콜/폴드/레이즈 (라운드당 각 1회)
- **폴드 3회 제한** + 전액 몰수 (AI도 동일 조건)
- **무승부 자연재해** — 직전 베팅액 2배 고정, 연속 무승부 시 기하급수적 증가
- **난이도 3단계** — 쉬움(AI 세트 공개) / 보통(AI 전체 카드 수 표시) / 어려움(낸 카드만 사후 공개)

## 트러블슈팅

### 화면 전환 불가 (가장 큰 버그)
`screen-setup` 안에 `.title-screen` 닫는 `</div>` 하나가 빠져서 모든 화면이 `screen-setup` 자식으로 들어가버림. 화면 전환 시 `display:none`이 전체에 적용돼 빈 화면이 됐던 것. `offsetParent: null` 단서로 찾아냄.

### 깃헙 배포 후 빈 화면
로컬에선 잘 되는데 깃헙에선 빈 화면 → 캐시 문제. `Ctrl+Shift+R` 강력 새로고침으로 해결.

### XML 선언 태그
세트 1 미니 SVG 안에 `<?xml version="1.0"?>` 가 남아있어서 HTML 파싱이 중간에 끊겼던 것.

### 폴드 순수익 표시 오류
팟 전액을 순수익으로 표시하고 있었음. 순수익 = 상대가 넣은 금액만으로 수정.

### AI 폴드 불평등
플레이어는 넣은 금액 전액 몰수인데 AI는 베팅 전 폴드 시 아무 페널티 없었음. AI도 폴드 전에 같은 금액을 팟에 먼저 넣도록 수정.

### 자연재해 배수 기준 오류
항상 10의 배수로만 계산됐던 것을 직전 실제 베팅액 기준으로 수정.
